# AI-Water-Demand-Forecasting
## Predictive Demand & Pumping Optimization

## Project Overview
This project addresses campus water shortages and high electricity costs by using **Agentic AI principles** and **XGBoost** to forecast daily water demand. It transforms raw IoT sensor data into an actionable dashboard for facility managers.

## Tech Stack
- **Language:** Python 3.11
- **Brain:** XGBoost (Regression)
- **Deployment:** Streamlit (Live Dashboard)
- **Data:** Pandas, NumPy, Scikit-Learn
- **Organization:** Jupyter Notebook (End-to-End Pipeline)

## Key Features
- **Temporal Analysis:** Integrated Academic Calendar (Exams/Vacations) and Weekend logic.
- **Weather Sensitivity:** Real-time temperature-based demand adjustment.
- **Cost Optimization:** "Pumping Recommendations" based on off-peak electricity tariffs.
- **Decision Support:** Automated alerts when tank levels are predicted to fall below 25%.

## Results
- **92% Accuracy** in predicting next-day consumption.
- Predicted **20% savings** in campus electricity bills through optimized pumping.
